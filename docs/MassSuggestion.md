Mass Suggestion
---------------

__Prerequisites:__ 21 or more ranks in a Perform skill, ability to cast arcane spells.

This ability functions like [*suggestion*](Suggestion), except that you can make the *suggestion* simultaneously to any number of creatures that you have already [fascinated](Fascinate). *Mass suggestion* is an enchantment (compulsion), mind-affecting, language-dependent ability.
