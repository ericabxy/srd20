### Rule

Every character level grants ~3 “virtual feat points”

### Caster Level Track

CL Scaling	Feat Cost
+½ level	0
+¾ level	1
+1/level	3

This is global, not per spell level.

### Archetypal Flavors (Constraint Packages)

These are:

- Qualitative
- Synergistic
- Balance-preserving
- Dangerous if freely recombined

Examples:

- Druid wild shape + companion + nature list
- Cleric domains + turning + divine list
- Paladin code + spell delay + smite suite
- Bard spell list + music + skills

> If removing a feature from a class would make the class stronger, that feature should not be optional.
